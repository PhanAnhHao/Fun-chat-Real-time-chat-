## 🚀 Setup & Run

**If first time installation or clone, install all packages first:**  
`npm i --save-exact`  

---

**Then, to run the app:**  
➡️ `cd fun-chat` *(if it is not in the `fun-chat` folder)*  
➡️ `npm start`  

---

**To run Firebase emulators:**  
➡️ `cd fun-chat` *(if it is not in the `fun-chat` folder)*  
➡️ `cd emulators`  
➡️ `npm run emulators`
